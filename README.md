# Fangirl Booth

A browser-based fan photobooth themed for The Runarounds. Visitors can start their camera or upload a photo, apply a stage-light filter (including Downtown and The Runarounds star treatment), choose encore stickers and palette doodles, capture a four-photo strip, and download the finished souvenir image.

## Files

- `index.html` – photobooth markup, controls, clickable sticker options, and doodle choices.
- `style.css` – responsive concert-poster, sticker, doodle, and palette styling.
- `script.js` – camera, upload, Downtown/star filters, stickers, palette doodles, strip rendering, and download behavior.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
